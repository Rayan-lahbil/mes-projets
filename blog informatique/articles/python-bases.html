Apprendre Python en partant de zéro

Python est le langage de programmation idéal pour débuter. Il est lisible, simple et utilisé partout : développement web, intelligence artificielle, analyse de données. Voici les bases essentielles.

Les variables

Une variable c'est une boîte qui stocke une valeur. On lui donne un nom et on lui assigne une valeur avec le signe =.

prenom = "Rayan"
age = 17
taille = 1.80
est_etudiant = True

Les conditions

Les conditions permettent d'exécuter du code selon une situation. On utilise if, elif et else.

age = 17

if age >= 18:
    print("Tu es majeur")
elif age >= 16:
    print("Tu es presque majeur")
else:
    print("Tu es mineur")

Les boucles

Les boucles répètent du code plusieurs fois. Il en existe deux types : for et while.

# Boucle for
for i in range(5):
    print("Ligne", i)

# Boucle while
compteur = 0
while compteur < 5:
    print("Compteur :", compteur)
    compteur += 1

Les fonctions

Une fonction est un bloc de code réutilisable. On la définit avec def et on l'appelle par son nom.

def saluer(prenom):
    print("Bonjour", prenom)

saluer("Rayan")
saluer("Marie")

Les listes

Une liste stocke plusieurs valeurs dans une seule variable.

langages = ["Python", "JavaScript", "HTML"]
print(langages[0])  # Python
langages.append("CSS")
print(len(langages))  # 4

Avec ces bases tu peux déjà créer des programmes complets en Python. La prochaine étape : créer ton premier jeu !
