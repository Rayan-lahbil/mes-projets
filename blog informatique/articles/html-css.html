HTML et CSS : créer sa première page web

HTML et CSS sont les deux langages fondamentaux du web. HTML structure le contenu, CSS le met en forme. Ensemble ils permettent de créer n'importe quelle page web.

C'est quoi HTML ?

HTML signifie HyperText Markup Language. C'est le squelette de toute page web — il définit les titres, paragraphes, images, boutons et liens.

<!DOCTYPE html>
<html>
<head>
    <title>Ma première page</title>
</head>
<body>
    <h1>Bonjour !</h1>
    <p>Bienvenue sur ma page web.</p>
    <a href="https://google.com">Aller sur Google</a>
</body>
</html>

Les balises HTML essentielles

Une balise HTML s'écrit avec des chevrons. La plupart ont une balise ouvrante et une balise fermante.

<h1>Titre principal</h1>
<h2>Sous-titre</h2>
<p>Un paragraphe de texte</p>
<img src="image.png" alt="description">
<button>Clique ici</button>
<div>Un conteneur</div>

C'est quoi CSS ?

CSS signifie Cascading Style Sheets. C'est la peinture de ta page — il gère les couleurs, tailles, polices et mises en page.

body {
    background-color: #1a1a2e;
    color: white;
    font-family: Arial, sans-serif;
}

h1 {
    color: #e94560;
    font-size: 36px;
    text-align: center;
}

button {
    background-color: #e94560;
    color: white;
    padding: 12px 24px;
    border-radius: 8px;
    border: none;
    cursor: pointer;
}

Relier HTML et CSS

Pour appliquer le CSS à ta page HTML, tu ajoutes une ligne dans le head de ton fichier HTML.

<link rel="stylesheet" href="style.css">

Avec HTML et CSS tu peux déjà créer des pages web complètes et bien designées. La prochaine étape : apprendre JavaScript pour rendre tes pages interactives !
