HTML et CSS : créer sa première page web

Dans ce tutoriel on va créer une carte de profil de A à Z. A la fin tu pourras la personnaliser avec ton propre nom, description, couleur etc

Voici ce qu'on va construire :

RL
Rayan Lahbil
Lycéen passionné par l'informatique à Montpellier

Étape 1 — Créer les fichiers

Dans VS Code, crée un dossier carte-profil avec deux fichiers dedans : index.html et style.css.

Étape 2 — La structure HTML

HTML c'est le squelette de la page : il dit ce qu'il y a sans s'occuper de l'apparence. Copie ce code dans index.html :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Ma carte de profil</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>

    <div class="carte">
        <div class="avatar">RL</div>
        <h1>Rayan Lahbil</h1>
        <p>Lycéen passionné par l'informatique — Montpellier</p>
        <button>Me contacter</button>
    </div>

</body>
</html>
CE QUE FAIT CHAQUE LIGNE

<div class="carte"> — un conteneur qui regroupe tout le contenu de la carte
<div class="avatar"> — le cercle avec les initiales
<h1> — le nom en grand
<p> — la description
<button> — le bouton cliquable

Si tu ouvres index.html dans Chrome maintenant tu verras le contenu mais sans aucun style : tout en noir et blanc, tout collé en haut à gauche. C'est normal, le CSS n'est pas encore là.

Étape 3 — Le fond de la page

Ouvre style.css et commence par styler le fond et centrer la carte :

body {
    background-color: #1a1a2e;
    display: flex;
    justify-content: center;
    align-items: center;
    min-height: 100vh;
    margin: 0;
}
CE QUE FAIT CHAQUE LIGNE

background-color — la couleur du fond de la page
display: flex — active le système de centrage
justify-content: center — centre horizontalement
align-items: center — centre verticalement
min-height: 100vh — la page prend toute la hauteur de l'écran

Étape 4 — La carte

Maintenant on style le rectangle principal :

.carte {
    background-color: #16213e;
    border-radius: 16px;
    padding: 40px 32px;
    width: 280px;
    text-align: center;
    border: 1px solid #2a4a6c;
}
CE QUE FAIT CHAQUE LIGNE

background-color — la couleur de fond de la carte
border-radius: 16px — arrondit les coins
padding — espace à l'intérieur de la carte
width — la largeur de la carte
text-align: center — centre tout le texte
border — une fine bordure autour de la carte

Étape 5 — L'avatar

On transforme le div en cercle coloré :

.avatar {
    width: 80px;
    height: 80px;
    background-color: #e94560;
    border-radius: 50%;
    display: flex;
    align-items: center;
    justify-content: center;
    font-size: 28px;
    font-weight: bold;
    color: white;
    margin: 0 auto 20px;
}
CE QUE FAIT CHAQUE LIGNE

width et height — taille du cercle (80px x 80px)
border-radius: 50% — transforme le carré en cercle parfait
display flex + align/justify center — centre les initiales dans le cercle
margin: 0 auto 20px — centre le cercle et ajoute un espace en dessous

Étape 6 — Le nom et la description

h1 {
    color: #ffffff;
    font-size: 20px;
    margin-bottom: 8px;
}

p {
    color: #a8a8b3;
    font-size: 14px;
    line-height: 1.5;
    margin-bottom: 24px;
}

Étape 7 — Le bouton

button {
    background-color: #e94560;
    color: white;
    padding: 10px 24px;
    border-radius: 8px;
    border: none;
    font-size: 14px;
    cursor: pointer;
}

button:hover {
    background-color: #c73652;
}
CE QUE FAIT CHAQUE LIGNE

border: none — supprime la bordure par défaut des boutons
cursor: pointer — le curseur devient une main au survol
button:hover — style appliqué quand la souris passe sur le bouton

Résultat final

Ouvre index.html dans Chrome et tu verras exactement la carte de profil affichée en haut de cet article. Tu peux maintenant la personnaliser : changer le nom, la description et les couleurs !

Tu viens d'apprendre les bases de HTML et CSS. Avec ces mêmes outils on peut créer des sites web complets.
